# AlgoRhythm — Algorithmic Rhythms for Renoise

AlgoRhythm is a Renoise `.xrnx` tool that generates algorithmic rhythms and melodic phrases using eight mathematical pattern generators — Euclidean, Bresenham, Cellular Automata, Markov chains, Logistic chaos, Perlin noise, weighted Random, and Straight subdivisions. Each voice lane gives you independent control over algorithm, steps, velocity, gate, pitch, ratchet, and micro-timing, all written directly into Renoise native phrases so patterns stay inside your project and loop without any external dependencies. Up to six voices can share a single instrument (useful for drum kits) or target separate instruments, with a built-in preset system and an Evolve mode that mutates patterns bar by bar for living, breathing grooves.
